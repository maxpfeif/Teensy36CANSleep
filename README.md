# Teensy36CANSleep
Deep sleep on Teensy 3.6 (Kinetis K66)

Using LLS (low-leakage stop) mode on the Teensy to save power when the CANbus goes idle.
Hopefully to be able to use the Teensy in an always powered state in a car, so it operates like any other module on the CANbus.
